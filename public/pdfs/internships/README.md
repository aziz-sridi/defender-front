# Internship PDF Ebooks

The internship topics currently link to the following Canva ebook:

- https://www.canva.com/design/DAG3plH_R_Y/r56DIPVXXT3vTjQcotEVUg/view?utm_content=DAG3plH_R_Y&utm_campaign=designshare&utm_medium=link2&utm_source=sharebutton

All “View PDF Ebook” buttons will open this link until individual PDFs are provided.

If you prefer separate PDF files for each topic, place them in this directory and update the `pdfUrl` values inside `src/lib/constants/jobs.ts`.

11. `marketing-automation.pdf` - Implementation of a Marketing Automation System

## Notes

- All PDF files should be placed in the `public/pdfs/internships/` directory
- File names should match exactly as listed above
- PDFs will be accessible at `/pdfs/internships/[filename].pdf`
- You can also use external URLs by updating the `pdfUrl` field in `src/lib/constants/jobs.ts`
